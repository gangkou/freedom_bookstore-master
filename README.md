全新版个人博客小程序，服务端使用的是Bmob后端云，无需开发服务端接口即可使用，快速便捷，代码简洁，功能包括文章列表，文章分类列表，赞赏功能，签到，收藏文章，查看文章，点赞文章，消息通知，评论文章，回复评论，可支持代码高亮，生成文章分享海报附带二维码等。本小程序有后台管理，可以发布文章，编辑文章，添加分类，管理分类，评论。可使用管理后台。
